# Favicon Setup Anleitung

Die folgenden Favicon-Dateien werden noch benötigt. Sie können diese mit Online-Tools erstellen:

## Benötigte Dateien:

1. **favicon-16x16.png** (16x16 Pixel)
2. **favicon-32x32.png** (32x32 Pixel)
3. **apple-touch-icon.png** (180x180 Pixel)
4. **favicon-192x192.png** (192x192 Pixel - für Android)
5. **favicon-512x512.png** (512x512 Pixel - für Android)
6. **og-image.jpg** (1200x630 Pixel - für Social Media Vorschau)

## Empfohlene Tools:

- **Favicon Generator**: https://realfavicongenerator.net/
- **Canva**: https://www.canva.com/ (für og-image.jpg)
- **Figma**: https://www.figma.com/ (für professionelle Designs)

## Designvorgaben:

- **Hauptfarben**:
  - Blau: #003A70
  - Orange: #F07E00
- **Logo**: Großes "R" oder vollständiges REKA-Logo
- **Hintergrund**: Gradient von Blau zu Orange oder einfarbig Blau

## Platzierung:

Alle Dateien sollten im `/public` Ordner liegen.

## OG-Image:

Das og-image.jpg sollte folgendes enthalten:
- REKA Logo
- Slogan: "Premium Industrieböden & Epoxidharzbeschichtungen"
- Professionelles Bild eines beschichteten Bodens
- Größe: 1200x630 Pixel (optimiert für Facebook, LinkedIn, etc.)
